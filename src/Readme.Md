1. Name of Project: DigiTools - Premium Digital Tools Platform. A modern React-based e-commerce website where users can browse, select, and purchase premium digital tools and resources.


2. Project Description: This project was developed as Assignment-6 for Programming Hero's MERN Stack course. DigiTools is a clean and user-friendly digital marketplace built with React.js. Users can explore various premium tools like AI writers, design templates, SEO toolkits, and more. The platform features a fully functional cart system, responsive design, and smooth user experience.

3. Technology: (a) React.js, (b) Tailwind CSS (c) React Icons (e) React Toastify.

4. Three Features about your project:
   a. Product Browsing & Filtering: Users can browse all premium digital tools with beautiful cards and filter between Products and Cart sections.

   b. Fully Functional Cart System: Add products to cart, remove individual items, view total price, and clear the entire cart with "Proceed to Checkout".

   c. Responsive Design and Toast Notifications:
   Fully responsive across all devices — mobile, tablet, and desktop with smooth animations and modern UI.Beautiful toast messages for add to cart, remove from cart, and checkout success using React Toastify.
